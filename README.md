### Adding, Removing, and Customizing Pages
All control of your web app can be found in `carbon.json`

Here is what the default `carbon.json` looks like:
```json
{
  "pages": [{
    "name": "Home",
    "doc": "index"
  },
  {
    "name": "About",
    "doc": "about"
  }],
  "logo": "carbon.png",
  "action": {
    "URL": "//carbon.toggled.tech",
    "text": "Try Carbon"
  },
  "title": "Carbon Starter"
}
```
